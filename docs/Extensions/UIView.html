  • addConstraint(item:attribute:relatedBy:toItem:attribute:multiplier:constant:)

    Helper for creating and adding NSLayoutConstraint but with default values provided.

    For instance

    addConstraint(item: view1, attribute: .CenterX, toItem: view2)

    is equivalent to

    addConstraint(
   NSLayoutConstraint(item: view1,
       attribute: .CenterX,
        relatedBy: .Equal,
        toItem: view2,
        attribute: .CenterX,
        multiplier: 1,
        constant: 0
   )
)

    Declaration

    Swift

    public func addConstraint(item view1: AnyObject,
                          attribute attr1: NSLayoutAttribute,
                          relatedBy: NSLayoutRelation = .equal,
                          toItem view2: AnyObject? = nil,
                          attribute attr2: NSLayoutAttribute? = nil,
                          multiplier: CGFloat = 1,

  • layout(_:)

    Lays out the views on both axis.

    Note that this is not needed for Horizontal only layouts.

    layout is primarily for laying out views vertically but horizontal statements are supported, making it perfect for describing a layout in one single statement.

    layout(
    100,
    |-email-| ~ 80,
    8,
    |-password-forgot-| ~ 80,
    >=20,
    |login| ~ 80,
    0
)
