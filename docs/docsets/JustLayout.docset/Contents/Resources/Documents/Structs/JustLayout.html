  • layout(_:visually:)

    Lays out the views on both axis.

    Note that this is not needed for Horizontal only layouts.

    layout:visually: is primarily for laying out views vertically but horizontal statements are supported, making it perfect for describing a layout in one single statement.

    JustLayout.layout(self.view, visually:
    100,
    |-emailView-| ~ 80,
    8,
    |-passwordView-forgotView-| ~ 80,
    >=20,
    |loginButton| ~ 80,
    0
    )

    Declaration

    Swift

    public static func layout(_ view: UIView, visually objects: Any...) -> [UIView]
